# WTF - Watch Those Friends

캐릭터를 생성하고 cctv로 관찰하자

## 대시보드
- [] 전체 CCTV 수
- [] 활성 CCTV 수
- [] Offline CCTV 수
- [] 현재 탐지 중인 캐릭터 수
- [] 오늘 발생한 이벤트 수
- [] 캐릭터 유형별 수
- [] 이벤트 유형별 통계


## CCTV 관제 화면
- [] CCTV 목록 조회
- [] CCTV별 화면 표시
- [] CCTV 화면 재생
- [] 캐릭터 포착 시 Bounding Box 오버레이
- [] 탐지된 캐릭터 정보 표시
- [] 이벤트 발생 시 화면 내 표시
- [] 이벤트 로그 클릭 시 해당 CCTV 화면으로 이동

### 초기 구현 방식
- [] 실제 스트리밍 대신 MP4 / Mock 화면 사용
- [] 탐지 결과는 랜덤으로 생성
- [] 영상 위에 좌표 기반 Bounding Box 표시

### 확장
- [] HLS 스트리밍
- [] WebRTC
- [] 실제 객체 탐지 결과 연동


## 지도 관제

## 캐릭터 생성 및 관리

- [] 자동차 (이름/색상/차종/성향)
